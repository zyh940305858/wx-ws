# ws-ws

个人使用的与微信hooks通信的WebSocket服务

## 微信hooks开放能力
🟢接收/发送文字  
🟢接收/发送表情  
🟢接收/发送图片  
🟢接收/发送APP  
🟢接收/发送语音  
🟢接收视频 ❌发送视频  
🟢发送/下载CDN图片  
🟢发送/下载CDN文件  
🟢拍一拍/收到拍一拍  
🟢邀请进群事件 ❌邀请成员入群  
🟢收到好友请求 ❌通过好友请求  
🟢撤回消息  
❌小程序
